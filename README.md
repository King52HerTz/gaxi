# 《轧戏》胡羞 & 肖稚宇 · 同人沉浸式空间

> **“我在现实中等你，并在摩天大楼顶端拥抱你。”**
>
> **“雪中容城，哪怕世界是假的，我的爱是真的。”**

这是一个专为电视剧《轧戏》粉丝打造的沉浸式同人互动空间。项目深度还原了剧中“现实（上海）”与“剧本（VR 容城）”双重世界的设定，通过细腻的交互与视觉转换，带你重温胡羞与肖稚宇（秦宵一）那场跨越次元的虐恋与救赎。

---

## 🎭 核心沉浸式体验

项目不仅是一个静态页面，更是一个可以互动的“数字剧本”：

- **� 现实与剧本的裂变**: 
  - 模拟 VR 设备接入感，一键在温润细腻的“上海现实”与冷峻肃杀的“容城剧本”间穿梭。
  - 整个空间的色彩、字体、氛围音轨乃至交互逻辑都会随之发生彻底演变。
- **� 平行时空通讯**: 
  - 还原剧中手机交互界面，那些在屏幕背后闪烁的真心、错过的解释，都在这里一一重现。
- **�📜 命运时间轴**: 
  - 从订婚宴的废墟到漫天飞雪的拥吻，复刻每一个命运转折点（Episode），带你重走他们的入局之路。
- **� 灵魂共振心跳**: 
  - 悬浮的心跳按钮，模拟肖稚宇那克制却热烈的心动。每一次点击，都能听见那些藏在冷酷外表下的、最笨拙的告白。
- **🎧 动态氛围音场**: 
  - 随着世界的切换，背景音乐将在现实的宁静与剧本的宏大间无缝衔接。

---

## 🔍 隐藏彩蛋 (Easter Eggs)

为了回馈深耕剧情的“入局者”，我们在空间中埋下了几处隐秘的彩蛋。只有足够细心的人，才能拼凑出完整的真相：

1. **碎片寻宝**: 
   - 页面中隐匿着 **4 处记忆碎片**：一副陈旧的 **VR 眼镜**、一张褶皱的 **旧图纸**、一片凋零的 **樱花瓣**、以及一簇不灭的 **火焰**。
   - 它们散落在不同的角落，有些只在特定模式下可见。
   - **集齐所有碎片后，将开启一封跨越时空的绝密信笺。**
2. **心跳回响**: 
   - 多次敲击右下角的心跳按钮，你会发现胡羞与肖稚宇内心深处从未说出口的私语。
3. **时空裂隙**: 
   - 尝试在不同的剧情阶段频繁切换世界，或许能触发不一样的系统反馈。

---

## 🛠️ 技术实现

- **前端框架**: Next.js 15 (Turbopack)
- **动效引擎**: Framer Motion (实现细腻的物理交互与转场效果)
- **视觉风格**: Tailwind CSS v4 + 响应式布局
- **交互设计**: 基于 React 19 的状态驱动，实现跨组件的模式同步

## 🚀 开启入局之旅

1. **克隆项目**
   ```bash
   git clone https://github.com/your-username/gaxi-pro.git
   cd gaxi-pro
   ```

2. **安装依赖**
   ```bash
   npm install
   ```

3. **启动开发服务器**
   ```bash
   npm run dev
   ```
   访问 `http://localhost:3000`，戴上耳机，即刻入局。
