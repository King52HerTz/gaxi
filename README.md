# 《轧戏》 | 胡羞 & 肖稚宇

> **“我在现实中等你，并在摩天大楼顶端拥抱你。”**
>
> **“雪中容城，哪怕世界是假的，我的爱是真的。”**

这是为胡羞和肖稚宇（秦宵一）做的一个小站。
想在这里，把那些没说完的话、没做完的梦，都留下来。

---

## ❄️ 关于这个空间

不仅仅是网页，更像是一个让我们重返“容城”的入口：

- **现实 vs 剧本**
  在这个页面，你可以随时“戴上”或“摘下”VR眼镜。
  看一看温暖喧闹的**上海现实**，或者切换回那个冰冷、精密却又刻骨铭心的**容城剧本**。

- **那些未发送的短信**
  还原了剧里的手机界面。那些他在屏幕背后删删减减的真心话，这次你可以看到了。

- **入局的时间轴**
  从订婚宴的一地鸡毛，到漫天飞雪的那个吻。每一个命运的转折点，都值得再走一遍。

- **听见他们的回响**
  页面右下角的心跳，藏着他们的过往。
  每一次点击，都是一句熟悉的**经典原声**。
  重温那些我们在屏幕前反复咀嚼过的对白——关于那场雪，关于那座楼，关于爱与被爱。

---

## 🔍 找找彩蛋

既然都“入局”了，不妨在页面里多逛逛，我藏了点东西：

1. **碎片收集**：我在角落里散落了 **4 个记忆碎片**（VR眼镜、旧图纸、樱花瓣、不灭的火）。有些碎片，只有在特定的世界里才能看见。
   *找齐它们，有一封跨越时空的信想给你看。*
2. **隐藏语音**：多戳几次心跳按钮，会有惊喜。
3. **世界裂缝**：试试快速切换现实与剧本，看看系统会不会“崩溃”。

## 🛠️ 技术栈
*也就是我是怎么把它造出来的*

- **Framework**: Next.js 15 (Turbopack)
- **Animation**: Framer Motion (为了那些丝滑的转场)
- **Style**: Tailwind CSS v4
- **State**: React 19 (用来同步两个世界的状态)

## 🚀 本地运行

如果你也想在本地跑起来：

1. **Clone**
   ```bash
   git clone [https://github.com/your-username/gaxi-pro.git](https://github.com/your-username/gaxi-pro.git)

```

2. **Install**
```bash
cd gaxi-pro
npm install

```


3. **Run**
```bash
npm run dev

```


打开 `http://localhost:3000`，戴上耳机，欢迎入局。
